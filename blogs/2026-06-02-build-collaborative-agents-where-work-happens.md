Microsoft announced expanded tools for developers to build AI agents in Teams, including general availability of the Teams SDK in Python, JavaScript, and C#. New interaction patterns such as targeted messaging, slash commands, and agent sessions enable agents to function as collaborative teammates within chats, channels, and meetings. Agent support now extends to shared and private channels, and sovereign cloud environments like GCC High and DoD gain full compatibility through the Microsoft 365 Agents Toolkit.
